# Foundations | Portfolio Website

A multi-page portfolio website built with raw HTML, CSS, and JavaScript. Showcases early front-end projects including a landing page, fansite, movie review app, platformer game, and design wireframes. Built as a student to learn the fundamentals of web development. Selected by faculty as the reference codebase for student projects in the cohort.

## Features
- **Multi-page routing**: Separate HTML pages for each project with consistent navigation.
- **Project showcase**: Individual pages for Landing Page, Fansite, Movietopia, Platformer Game, and Wireframe.
- **Contact section**: Email link and profile information for internship opportunities.
- **Clean layout**: Minimal design focused on project presentation and readability.

## Technologies Used
- **HTML**: Semantic markup for structure and accessibility
- **CSS**: Custom styling with responsive layouts
- **JavaScript**: Interactive elements and page routing

## Project Structure
```

```

### Navigation Structure
The site links to individual project pages from the home page and projects page. Each project page includes a description, technologies used, and links back to the main site.

### Key Pages
- **index.html:** Profile introduction with contact information and links to projects.
- **projects.html:** Gallery-style overview of all projects.
- **landing-page.html:** Single-page marketing layout built to master box-model fundamentals.
- **fansite.html:** Multi-page site demonstrating custom CSS Grid systems.
- **reviews.html:** PHP/MySQL application with CRUD and session-based authentication.
- **platformer.html:** 2D Java game with collision detection and game loops.
- **wireframe.html:** 154-screen high-fidelity mobile prototype.
- **design.html:** Collection of digital assets and layouts using Adobe Creative Cloud.
